# 设计文档索引 (Design Index)

本文档作为 `claude_proxy` 的架构设计索引，记录了各模块的设计决策。

## 模块列表

### 1. [整体架构与流程](./architecture.md)
描述系统的整体拓扑结构、请求/响应的生命周期以及数据流向。

### 2. [协议转换引擎](./transformer.md)
核心模块，详细说明如何将非标准的文本输出转换为标准的 Anthropic 工具调用块。

### 3. [流式处理器 (SSE Stream Handler)](./stream_handler.md)
描述如何在 SSE 流中实时处理数据包，以及如何处理被切断的工具调用 JSON 片段。

### 4. [配置与插件系统](./config_system.md)
说明如何通过环境变量映射模型，以及如何扩展支持不同厂商的特定输出格式。

## 核心设计准则
- **协议优先**：严格遵循 Anthropic Messages API 规范。
- **无状态设计**：代理本身不存储会话状态，完全依赖请求中的 context。
- **高性能正则表达式**：使用高效的解析算法识别文本中的工具指令。
